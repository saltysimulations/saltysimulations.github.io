Installation

Stable or development version?

When installing the 74S, you can choose between two different versions. The stable version has less features than the development version, while also being generally less prone to bugs. We recommend the development version if you want to make use of the latest and greatest features.

FlyByWire Installer​

We recommend installing via the FlyByWire Installer, which provides a seamless and easy-to-use way to install and keep the Salty 74S updated. Simply choose your preferred version and install.

Alternative Methods​

caution

To avoid issues with installation, we highly recommend using the FlyByWire installer instead of downloading directly.

Manual installation is possible by downloading either the stable or development versions and copying the contents of the archive into your Community folder. Depending on the platform you bought Microsoft Flight Simulator on, the location of this folder will differ.

  • Microsoft Store
  • Steam
  • Boxed
C:\Users\username\AppData\Local\Packages\Microsoft.FlightSimulator_[]\LocalCache\Packages\Community
C:\Users\username\AppData\Roaming\Microsoft Flight Simulator\Packages\Community
C:\Users\username\AppData\Local\MSFSPackages\Community
Note

If you customized the install location when installing MSFS, regardless of platform, your Community folder will be located wherever you installed it.
